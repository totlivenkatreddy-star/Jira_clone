# Jira Clone - Project Management System

A Jira-like project management web application built to manage projects, issues, tasks, comments, and team collaboration.

This project is developed to understand and implement real-world backend architecture, REST APIs, database design, authentication, and full-stack application development.

---

## 🚀 Project Overview

Jira Clone allows users to:

- Create and manage projects
- Create and track issues/tasks
- Assign tasks to team members
- Manage issue status workflow
- Add comments and discussions
- Track project progress
